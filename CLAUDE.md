# CLAUDE.md

Guidance for Claude Code when working in this repository.

## What this is

Marketing website for **Matari Solutions LLC**, a mission-driven government
contracting firm. It's a **static site** generated by **Gulp** from Markdown +
front-matter pages wrapped in a shared **EJS** layout. The visual design is
built on a Bootstrap 5 + Font Awesome 6 front-end, branded for Matari Solutions.

The companion brand/style guide is **`design.md`** — read it before changing
colors, type, components, copy, or imagery.

## Commands

```bash
npm run dev        # gulp serve: build + BrowserSync live-reload (localhost:3000)
npm run build      # clean + production build into dist/
npm run optimize   # sharp: resize/compress images in src/assets/img (run after adding images)
npm run lint       # eslint (build tooling only)
npm run format     # prettier --write
npm run format:check
```

CI (`.github/workflows/deploy.yml`) runs `lint` → `format:check` → `build` on
push to `main`, then deploys `dist/` to GitHub Pages. **Lint and format:check
must pass or the deploy fails** — run `npm run format` and `npm run lint` before
pushing.

## How the build works (`gulpfile.js`)

- **html**: `src/pages/*.md` → gulp-front-matter → gulp-markdown → gulp-layout.
  Front-matter fields (`title`, `description`, `layout`, `noindex`) plus a
  derived `page` path become the EJS template locals; the page body is injected
  as `<%- contents %>`.
- **styles**: all CSS concatenated + minified → `dist/assets/css/app.min.css`
  (order in `CSS_BUNDLE`; `custom.css` last so overrides win). `rebase:false`
  keeps Font Awesome's `../fonts/...` URLs valid.
- **scripts**: all JS concatenated + minified → `dist/assets/js/app.min.js`
  (order in `JS_BUNDLE`, jQuery first, `main.js` last).
- **assets**: fonts/images/favicons copied verbatim (bundled CSS/JS excluded).
- **waveScript**: `wave.js` copied unbundled — it's loaded inline on the home
  page for the hero wave effect.
- **staticFiles**: `robots.txt`, `sitemap.xml`.

## Conventions & gotchas

- **Edit `src/`, never `dist/`** (generated) and never the vendor
  `src/assets/css/style.css` (huge, Prettier-ignored). Site-specific CSS goes in
  **`src/assets/css/custom.css`**, which is bundled last.
- **Brand colors** are CSS variables overridden in `custom.css`:
  `--theme-color: #3fded9` (teal accent) and `--theme-color2: #173460` (navy).
  These drive every gradient and accent. See `design.md`.
- **Markdown pages contain raw HTML** (theme components). Keep each top-level
  HTML block contiguous (no blank lines inside a block; one blank line between
  blocks) so markdown passes it through untouched.
- **Pages**: `index` (Home Two layout), `about`, `services`, `contact`,
  `thankyou` (noindex), `404` (noindex). Nav lives in `src/layouts/default.ejs`.
- **Contact form** uses Netlify Forms (`data-netlify`, posts to `thankyou.html`).
- **Images**: after adding to `src/assets/img/`, run `npm run optimize`. Large
  content photos (`normal/*.png`) should be resized to display size.
- **SEO**: canonical/OG/Twitter/JSON-LD are built in the layout head from
  `SITE_URL` (currently `https://www.matarisolutions.com` — update if the domain
  differs). `sitemap.xml` and `robots.txt` live in `src/`.
- Some theme placeholder photos (`normal/about1-1.png`, `bg/breadcrumb-bg.png`)
  are gray "W×H" placeholders awaiting real imagery.
